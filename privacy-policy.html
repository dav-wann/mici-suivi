Politique de confidentialité

MICI Suivi  ·  Dernière mise à jour : 30 mai 2026

MICI Suivi ne collecte, ne vend et ne partage aucune donnée personnelle de santé. Toutes vos données sont stockées localement sur votre appareil.

1. Qui sommes-nous

MICI Suivi est une application mobile de suivi personnel développée à titre indépendant. Contact : micisuivi.app@gmail.com

2. Données stockées sur votre appareil

L'ensemble des données suivantes est stocké uniquement sur votre téléphone, via le stockage local de l'application (AsyncStorage) :

  • Selles enregistrées (date, heure, présence de sang)
  • Repas journalisés (description, photo, résultat d'analyse IA)
  • Poids quotidiens
  • Taille et profil
  • État clinique (RAS / rémission / poussée)

Ces données ne quittent jamais votre appareil, sauf dans les deux cas décrits ci-dessous.

3. Analyse IA — données transmises au serveur

Lorsque vous déclenchez une analyse IA (analyse d'un repas ou analyse statistique des selles), les données suivantes sont envoyées à notre serveur proxy sécurisé, hébergé sur Railway (Europe) :

  • Analyse repas : description textuelle du repas, photo optionnelle, état clinique sélectionné
  • Analyse des habitudes : statistiques agrégées des 30 derniers jours (nombres de selles, fréquences — sans horodatage précis)

Ces données sont transmises à l'API Anthropic (Claude) pour générer l'analyse, puis la réponse est renvoyée à votre application. Aucune de ces données n'est conservée sur notre serveur. La transmission est chiffrée (HTTPS).

Anthropic applique sa propre politique de confidentialité, disponible sur anthropic.com/privacy.

4. Export PDF

La fonction d'export génère un fichier PDF à partir de vos données locales. Ce fichier est créé sur votre appareil et partagé uniquement si vous choisissez de le partager manuellement. Aucun PDF n'est envoyé à nos serveurs.

5. Pas de compte, pas de tracking

MICI Suivi ne requiert aucun compte utilisateur, aucune inscription, aucune adresse e-mail. L'application ne contient aucun outil d'analyse comportementale, aucune publicité, aucun SDK de tracking tiers.

6. Permissions Android

  • Caméra : uniquement pour photographier un repas, si vous le demandez
  • Stockage / médias : pour sélectionner une photo depuis la galerie

Ces permissions ne sont utilisées qu'à votre initiative et ne donnent accès à aucune autre donnée de votre appareil.

7. Données des mineurs

MICI Suivi n'est pas destinée aux enfants de moins de 16 ans. Nous ne collectons sciemment aucune donnée provenant de mineurs.

8. Vos droits

Toutes vos données étant stockées localement, vous pouvez les supprimer à tout moment en désinstallant l'application. Pour toute question, contactez : micisuivi.app@gmail.com

9. Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut de cette page.
